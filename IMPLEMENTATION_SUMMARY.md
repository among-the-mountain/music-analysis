## API端点列表

1. `GET /api/status` - 数据状态
2. `GET /api/music-type-distribution` - 音乐类型分布
3. `GET /api/album-type-analysis` - 专辑类型分析
4. `GET /api/album-type-top10` - 专辑TOP10
5. `GET /api/publish-trend` - 发布趋势
6. `GET /api/top-artists` - TOP作者
7. `GET /api/wordcloud` - 词云图
8. `GET /api/sentiment-trend` - 情感分析
9. `GET /api/reload` - 重新加载数据

## 数据结构

### 网易云音乐数据字段
```
- song_id: 歌曲ID
- song_name: 歌曲名称
- artist_name: 作者名称
- album_name: 专辑名称
- album_type: 专辑类型
- music_type: 音乐类型
- duration_ms: 时长
- popularity: 人气值
- publish_year: 发布年份
- publish_date: 发布日期
```
